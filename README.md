# pomomo

A desktop Pomodoro planner built with Electron. You can create task and break blocks, customize their durations, and drag them into the order you want for your session flow.

## Features
- Custom task blocks with adjustable durations
- Custom break blocks with adjustable durations
- Draggable queue blocks for reordering your session
- Start, pause, and reset controls for the active timer

## Run locally
1. Install dependencies with `npm install`
2. Start the app with `npm start`

## Project structure
- `main.js` – Electron window bootstrap
- `renderer.js` – timer and queue behavior
- `index.html` – app layout
- `styles.css` – visual styling
